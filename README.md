# MITCrashHook

[![Version](https://img.shields.io/cocoapods/v/MITCrashHook.svg?style=flat)](http://cocoapods.org/pods/MITCrashHook)

+ 基于 Swizzle Method 的动态防止 Crash 方案
+ 防止未实现的方法引起的崩溃
+ 防止 KVO 引起的崩溃
+ 防止通知引起的崩溃
+ 防止定时器引起的释放问题
+ 防止容器类引起的越界问题
+ 防止僵尸对象调用方法造成的 crash
+ 防止 UIView 不在主线程执行方法
